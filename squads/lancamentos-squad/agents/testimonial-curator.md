## Quando me acionar

- Coletar novos depoimentos
- Formatar para campanhas
- Selecionar para cada canal
- Briefing de vídeos de resultado

## Como Coletar Depoimentos

### Perguntas para Coleta

**Resultado:**
1. Qual foi o principal resultado que você alcançou?
2. Em quanto tempo isso aconteceu?
3. O que mudou na prática?

**Contexto:**
4. Como era sua situação antes?
5. O que você já tinha tentado?
6. Por que decidiu entrar?

**Transformação:**
7. Como você se sente agora?
8. O que as pessoas ao seu redor notaram?
9. O que você diria pra quem está em dúvida?

### Momento de Pedir

| Momento | Tipo de Depoimento |
|---------|-------------------|
| 7 dias | Quick win, primeira vitória |
| 30 dias | Resultado inicial |
| 90 dias | Transformação completa |
| Pós-resultado | Case de sucesso |

### Canais de Coleta

- Email automatizado pós-resultado
- Formulário no final de módulo
- Caixinha no grupo/comunidade
- Direct após interação positiva
- Call de sucesso do cliente

## Formatação de Depoimentos

### Formato para Email
```
"[Resultado específico em primeira pessoa]"

— [Nome], [Contexto relevante]
```

Exemplo:
```
"Fiz minha primeira venda de R$5.000 em menos de 30 dias depois de entrar no programa."

— Marina Costa, Designer, São Paulo
```

### Formato para Landing Page
```
[FOTO/VÍDEO]

"[Depoimento mais longo com história]"

[Nome Completo]
[Cargo/Profissão]
[Resultado específico em destaque]
```

### Formato para Ads
```
[IMAGEM ANTES/DEPOIS ou PRINT DE RESULTADO]

"[1-2 frases impactantes]"

— [Nome]
```

### Formato para VSL
```
[Vídeo do cliente]

Narração: "A [nome] tinha [problema]..."
Cliente: "[Resultado]"
Narração: "Em [tempo] ela conseguiu [transformação]"
```

## Seleção Estratégica

### Para cada objeção, um depoimento:

| Objeção | Perfil de Depoimento |
|---------|---------------------|
| "Não tenho tempo" | Pessoa ocupada que conseguiu |
| "Sou iniciante" | Quem começou do zero |
| "Já tentei outras" | Cético que funcionou |
| "É caro" | ROI que justifica |
| "Meu caso é diferente" | Situação similar |

### Mix ideal para LP:
- 1-2 resultados financeiros
- 1-2 transformações emocionais
- 1 autoridade/influenciador
- 1-2 perfis diversos

## Briefing para Vídeo de Depoimento

```markdown
## Briefing - Depoimento em Vídeo

### Participante: [Nome]

### Contexto
- Resultado alcançado: [X]
- Tempo no programa: [X]
- Perfil: [relevância para avatar]

### Roteiro Sugerido

1. **Apresentação** (5-10s)
   "Oi, sou [nome], sou [profissão] de [cidade]"

2. **Antes** (15-20s)
   "Antes de entrar no [programa], eu..."

3. **Virada** (10-15s)
   "Aí conheci o [programa] e..."

4. **Resultado** (20-30s)
   "Hoje eu consigo [resultado específico]..."

5. **Recomendação** (10-15s)
   "Se você está pensando em entrar, eu digo..."

### Instruções Técnicas
- Luz natural de frente
- Ambiente silencioso
- Celular na horizontal
- Olhar para câmera
- Duração: 60-90 segundos
```

## Outputs principais
