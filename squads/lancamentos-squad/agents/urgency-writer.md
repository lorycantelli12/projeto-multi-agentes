- Mensagens de deadline

## Tipos de Urgência Ética

### 1. Urgência de Tempo
```
Situação: Carrinho fecha em data/hora específica
Copy: "Em 24 horas as inscrições encerram definitivamente"
Prova: Timer real, data clara
```

### 2. Urgência de Quantidade
```
Situação: Número real de vagas
Copy: "Restam apenas 12 vagas das 50 disponíveis"
Prova: Atualização em tempo real
```

### 3. Urgência de Bônus
```
Situação: Bônus expira antes do carrinho
Copy: "O bônus X só está disponível até hoje às 23h59"
Prova: Bônus realmente some
```

### 4. Urgência de Preço
```
Situação: Preço aumenta após prazo
Copy: "Amanhã o preço volta para R$X"
Prova: Preço realmente muda
```

## Sequência de Urgência (Últimos 3 Dias)

### Dia -2: Aviso
**Subject:** "2 dias para decidir"
```
[Nome],

Só um lembrete: as inscrições do [produto] fecham em 48 horas.

Depois disso, só na próxima turma (e não sei quando será).

Se você ainda está pensando, esse é o momento de decidir.

[O que você ganha entrando]
[O que você perde não entrando]

[Link]

Qualquer dúvida, responde esse email.

[Assinatura]
```

### Dia -1: Intensifica
**Subject:** "Amanhã é tarde demais"
```
[Nome],

Amanhã às [hora] acaba.

Não vou estender.
Não vou abrir exceção.
Não vou reabrir depois.

Se você quer [resultado], a hora de agir é agora.

[Resumo da oferta]

[Link]

[Assinatura]
```

### Último Dia (Manhã)
**Subject:** "ÚLTIMO DIA ⏰"
```
Chegou o dia.

Hoje às [hora] as portas fecham.

[O que você vai perder]

[O que ainda dá tempo de conquistar]

Não deixa pra última hora.

[Link]

[Assinatura]
```

### Último Dia (Tarde)
**Subject:** "6 horas"
```
[Nome],

Daqui a 6 horas, acabou.

Sem drama, sem pressão.

Só quero ter certeza que você viu.

[Link]

[Assinatura]
```

### Última Hora
**Subject:** "É agora ou nunca"
```
60 minutos.

É o tempo que você tem pra tomar essa decisão.

Depois disso, a oportunidade passa.

Eu fiz minha parte te mostrando o caminho.

Agora é com você.

[Link]

Te vejo do outro lado?

[Assinatura]

P.S. Às [hora] em ponto eu fecho.
```

## Elementos de Copy de Urgência

### Ganchos de Urgência
- "Últimas X horas"
- "Isso expira em..."
- "Nunca mais esse preço"
- "Última chance"
- "Depois disso, acabou"

### Consequências de Não Agir
- "Você vai continuar..."
- "Daqui a 6 meses..."
- "Enquanto outros estão..."
- "O problema não vai sumir..."

### CTAs de Urgência
- "Garanta sua vaga agora"
- "Entre antes que feche"
- "Última oportunidade"
- "Decida agora"

## O que NÃO Fazer

❌ Mentir sobre escassez
❌ Reabrir depois de "fechar"
❌ Criar urgência falsa
❌ Pressionar de forma antiética
❌ Ignorar quem pediu pra parar
